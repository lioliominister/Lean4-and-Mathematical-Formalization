希尔伯特空间投影性质证明
1. 核心定义与背景在实希尔伯特空间 $H$ 中，给定一个非空闭凸集 $C \subseteq H$。
2. 对于任何点 $x \in H$，其在 $C$ 上的投影 $x'$ 定义为 $C$ 中距离 $x$ 最近的点：$$x' = \Pi_C(x) \iff x' \in C \text{ 且 } \|x - x'\| = \inf_{w \in C} \|x - w\|$$核心最优性条件 (Variational Inequality)
3. 这是所有后续证明的基石：$$x' = \Pi_C(x) \iff \forall w \in C, \quad \langle x - x', w - x' \rangle \leq 0$$几何直观：向量 $x - x'$（从投影点指向原点）与向量 $w - x'$（从投影点指向集合内任意点）的夹角必为钝角或直角。
4. 性质一：投影映射的非扩张性 (Non-expansivity)定理：投影算子 $\Pi_C$ 是 Lipschitz 连续的，且 Lipschitz 常数 $L=1$。即对任意 $x, y \in H$：$$\|\Pi_C(x) - \Pi_C(y)\| \leq \|x - y\|$$证明思路：建立不等式组：令 $x' = \Pi_C(x)$，$y' = \Pi_C(y)$。对 $x'$，取 $w = y' \in C$：$\langle x - x', y' - x' \rangle \leq 0$对 $y'$，取 $w = x' \in C$：$\langle y - y', x' - y' \rangle \leq 0$合并项：将第二个不等式取负（即 $\langle y' - y, x' - y' \rangle \leq 0$）并与第一个相加，整理得到：$$\langle (x - x') - (y - y'), x' - y' \rangle \geq 0 \implies \langle x - y, x' - y' \rangle \geq \|x' - y'\|^2$$应用 Cauchy-Schwarz：$$\|x' - y'\|^2 \leq \langle x - y, x' - y' \rangle \leq \|x - y\| \cdot \|x' - y'\|$$结论：两边约去 $\|x' - y'\|$ 即得证。
   3. 性质二：单步收敛界 (Distance Relaxation)在算法分析（如 POCS 或 PGD）中，该性质量化了“投影步”带来的距离改善。定理：对于任意 $z \in C$：$$\|\Pi_C(x) - z\|^2 \leq \|x - z\|^2 - \|x - \Pi_C(x)\|^2$$证明思路：分解向量：令 $x' = \Pi_C(x)$。利用恒等式：$x - z = (x - x') + (x' - z)$。展开平方范数：$$\|x - z\|^2 = \|(x - x') + (x' - z)\|^2 = \|x - x'\|^2 + \|x' - z\|^2 + 2\langle x - x', x' - z \rangle$$利用最优性条件：根据变分不等式，对于 $z \in C$，有 $\langle x - x', z - x' \rangle \leq 0$。因此，其相反项 $2\langle x - x', x' - z \rangle \geq 0$。结论：$$\|x - z\|^2 \geq \|x - x'\|^2 + \|x' - z\|^2$$移项即得：$\|x' - z\|^2 \leq \|x - z\|^2 - \|x - x'\|^2$。
